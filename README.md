**sh-conv** is a self-hosted website/app that lets you convert files to different formats, as well as download media from a big variety of applications

## Features
- Convert audio, image and videos to another format easily.
- Download videos from [1000+ sites](https://kolioaris.xyz/docs/sh-conv/supported-download-sites).
- Different themes & color themes.
